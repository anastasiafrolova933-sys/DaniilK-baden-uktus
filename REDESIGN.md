# Redesign — тестовые прототипы

Ветка для прототипов нового дизайна сайта комплекса. **Не сливать в main без команды.** Боевой `index.html` в этой ветке не тронут, прототипы лежат в отдельных подпапках.

## Прототипы

### v2 — Editorial Brutalism (архив)
- Файл: `archive/v2-editorial-brutalism/index.html`
- Концепт: огромный uppercase-гротеск (Inter Tight 800), курсивный акцент (Fraunces italic), кислотный лайм `#c7ff3a`, кастомный курсор с `mix-blend-mode: difference`, Three.js liquid blob в hero, маркизы, прелоадер.

### v3 — Cinematic Journey (текущий)
- Файл: `v3-cinematic/index.html`
- Концепт: фиксированный Three.js канвас + кинематографичный скролл сквозь 7 «глав». Камера летит по z-станциям, окружение трансформируется (туман, цвет, ember-light). 3D-объекты в каждой главе: liquid blob → water surface → caustics corridor → ember field → tariff crystals → combo torus rings → glowing portal CTA. Прогресс-рельс справа, HUD по углам с координатами Уктуса.
