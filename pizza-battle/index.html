Формат

Три этапа забега

Каждый игровой день — с 14:00 до 16:00.

01

Разминка

Лёгкие конкурсы на скорость реакции для настроя.

02

Эстафета

Командный этап — передача куска пиццы через препятствия.

03

Скоростное поедание

Главный забег: кто быстрее съест порцию — 30 см, 8 кусков. Побеждает лучшее суммарное время.

Важно знать

Условия

Возраст
Только для совершеннолетних (18+) без медицинских противопоказаний к употреблению пиццы в больших объёмах.
Стоимость
Для участников — бесплатно. Группа поддержки оплачивает вход по тарифу дня.
Состав
Пицца стандартная, без копчёностей, орехов и других аллергенов. При аллергии — предупредите организаторов.
Отбор
Регистрация свободная. Из всех заявок мы случайно выберем участников и свяжемся по телефону.
